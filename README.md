# DoneFlow

A clean, minimal to-do app with persistence, swipe-to-delete and dark mode.

![DoneFlow Demo](screenshots/demo.gif)

## Features
- Add / Delete tasks
- Mark as complete
- Swipe to delete
- Data saved locally (`shared_preferences`)
- Clean, animated UI

## Tech
- Flutter
- `shared_preferences`
- `Dismissible` widget

## Run
```bash
flutter pub get
flutter run
